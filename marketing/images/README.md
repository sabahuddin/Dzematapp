# Slike za DžematApp Landing Page

Ovdje trebaju biti sljedeće slike:

## Obavezne slike

| Naziv fajla | Dimenzije | Opis |
|-------------|-----------|------|
| logo-placeholder.png | 200x60px | Logo DžematApp |
| og-image.png | 1200x630px | Open Graph slika za social sharing |
| favicon.png | 32x32px | Favicon |
| hero-screenshot.png | 1200x800px | Screenshot aplikacije za Hero sekciju |

## Kako zamijeniti placeholder

1. Napravi screenshot aplikacije (dashboard ili glavni ekran)
2. Exportuj logo iz dizajna
3. Kreiraj favicon (mali logo)
4. Napravi OG sliku (logo + tekst za social media)

## Nakon dodavanja slika

U index.html zamijeni:
- `.screenshot-placeholder` div sa `<img src="images/hero-screenshot.png">`
- `images/logo-placeholder.png` sa pravim logom
